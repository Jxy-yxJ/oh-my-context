## 控制流：审核流程

```
┌─────────────┐     ┌─────────────┐     ┌─────────────────────────────────────┐
│   suggest   │────▶│  AI Extract │────▶│   Create Candidate                  │

└─────────────┘     └─────────────┘     │   • confidence_score                │
                                        │   • source_session                  │

                                                       │

                                        ┌─────────────────────────────────────┐

                   ┌──────────────────┐    ┌──────────────────┐    ┌──────────────────┐
                   │ confidence > 0.9 │    │   auto_promote   │    │  manual review   │
```

## 对比：v1 vs v2

### v1 (当前)
```
suggest ──▶ create memory ──▶ push
              │
              ▼
         ┌─────────┐
         │ memory/ │  (所有内容混在一起)
         │ (混乱)  │
         └─────────┘
```

### v2 (升级后)
```
suggest ──▶ create candidate ──▶ review ──▶ promote ──▶ memory
                                    │
                                    ▼
                              ┌───────────┐
                              │ candidate/│  (待审核)
                              │ (pending) │
                              └───────────┘
                                    │
                                    ▼
                              ┌───────────┐
                              │ memory/   │  (分层管理)
                              │ ├── core/ │
                              │ ├── active│
                              │ └── archive│
                              └───────────┘
```
